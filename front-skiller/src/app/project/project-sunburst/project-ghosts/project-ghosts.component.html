Handling the ghosts


The table below lists the unregistered contributors picked up for this project.
Each Pseudo (on the first column), is not recognized in any staff member.
In order to complete the evaluation, all of these "commit/pseudo" need to be revoked or assigned.

3 actions are possible :

  • Type a Related login, to be associated with this pseudo. A single developer might have multiple pseudos.
  • Tag this pseudo as Technical. There is nobody behind this pseudo.
  •  Create a new staff member with this pseudo.
